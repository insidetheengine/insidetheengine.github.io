Política de Privacidade

Última atualização: 17 de Abril de 2026

A Inside the Engine Media valoriza a sua privacidade. Esta política descreve como tratamos as informações ao utilizar a API do TikTok para publicação de conteúdo.

1. Coleta de Informações

Nosso aplicativo utiliza apenas as permissões necessárias para realizar o upload de vídeos gerados pelo nosso sistema para a conta do usuário autenticado. Não coletamos nem armazenamos dados pessoais de terceiros.

2. Uso dos Dados

Os tokens de acesso são utilizados exclusivamente para o processo de publicação e são armazenados localmente de forma segura.

3. Compartilhamento

Não compartilhamos, vendemos ou alugamos informações de usuários para terceiros.
